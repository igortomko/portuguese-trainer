# 🇧🇷 Português B2 — Тренажёр

Персональный тренажёр португальского языка для Игоря.

## Deploy

Deployed on Vercel. Env variable required:
- `LINEAR_API_KEY` — для автоматической отметки задач Linear как Done при 100% прогрессе

## Features

- 19 тем курса до B2
- 3 режима: карточки, квиз, список
- Прогресс сохраняется в localStorage
- Автоматическое закрытие Linear-задач при 100%
- Озвучка португальского произношения
